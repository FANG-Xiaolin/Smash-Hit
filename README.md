# Smash-Hit
- A game on PC
- OpenGL and C++
- Course project for Computer Graphics by Xiaolin Fang, Guangchen Lin, and Shuyue Lai
- Click to throw the ball and break the barriers. W/A/S/D to navigate the map.
- Life -1 if hit by something.
